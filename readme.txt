梧桐雨工具箱 Ver 1.2
1，修正无法查询手机号码归属地，whois查询，原因是因为公开的key使用被封禁了，我自己申请了一枚，暂时能用，未来出现bug会及时进行调整。欢迎反馈bug。

梧桐雨工具箱 Ver 1.1
1，新增了cookie查看功能，在添加cookie的地方点击获取当前浏览器所有cookie即可读取到当前浏览器所有cookie。
2，新增了手机号码归属地查询功能，方便对手机号码的查询（支持联通移动电信）。
3，新增短网址转换功能，方便一键转换短网址，不需要再去跑到特殊网站转换了。
4，删除了清除cookie功能，该功能在上个版本存在bug，调用了一个不存在的接口。
